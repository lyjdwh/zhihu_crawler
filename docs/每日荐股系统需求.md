# 每日股市推荐系统 - 需求文档

## 一、系统目标

基于知乎股市专家的历史经验和当日分析，结合宏观经济数据，为投资者提供每日股市操作建议。

---

## 二、数据来源

### 2.1 专家列表（可扩展）

| 专家 | 知乎ID | 特点 |
|------|--------|------|
| 奥特之父 | xu-ze-qiu | 金融界门德尔松，实战派，擅长宏观分析和中线操作 |
| MR Dang | mr-dang-77 | 独立投资人，擅长价值投资和股息策略 |

**扩展说明**：后续可增加更多专家，需要修改 `config/experts.yaml` 配置文件。

### 2.2 数据维度

#### a) 专家历史经验
- 历史回答中的投资理念和技巧
- 擅长分析的股票类型和行业
- 操作风格（短线/中线/长线）
- 成功的投资案例

#### b) 专家当日回答
- 当日对股市行情的分析和判断
- 当日提到的股票或行业
- 当日的操作建议

#### c) 公司情况
- 当前股价
- 公司基本面（营收、利润、估值）
- 近期消息和公告
- 机构评级

#### d) 宏观情况
- A股整体走势
- 宏观经济数据（GDP、CPI、PPI、PMI等）
- 政策动向
- 国际市场（美股、港股、汇率、大宗商品）

## 三、系统功能

### 3.1 数据采集模块
- [ ] 自动爬取指定专家的最新回答
- [ ] 支持日期范围筛选
- [ ] 支持主题关键词筛选（金融/股市/投资）
- [ ] 自动获取完整回答内容

### 3.2 数据分析模块
- [ ] 提取当日回答中的关键信息
- [ ] 识别提到的股票和行业
- [ ] 分析情感倾向（看多/看空/中性）
- [ ] 提取操作建议

### 3.3 宏观数据模块
- [ ] 接入A股行情数据
- [ ] 接入宏观经济数据（可选）
- [ ] 计算市场情绪指标

### 3.4 推荐生成模块
- [ ] 综合多方观点生成推荐
- [ ] 给出具体操作建议（买入/卖出/持有）
- [ ] 标注风险提示
- [ ] 生成每日报告

## 六、使用说明

### 6.1 数据采集
```bash
# 爬取专家最新回答
python scripts/crawl_user.py --user xu-ze-qiu --count 10 --topic finance
python scripts/crawl_user.py --user mr-dang-77 --count 10 --topic finance

# 按日期筛选
python scripts/crawl_user.py --user xu-ze-qiu --after-date 2026-03-01 --topic finance
```

### 6.2 生成报告
运行推荐脚本（需要实现），生成每日报告：
```bash
python scripts/daily_report.py --date 2026-03-15
```

## 八、注意事项

1. **风险提示**：本系统仅供参考，不构成投资建议
2. **数据更新**：每日需要更新最新数据
3. **专家选择**：可根据个人偏好调整专家列表
4. **合规要求**：遵守知乎API使用规范，不要过于频繁请求
